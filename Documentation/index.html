Prerequisites

Please ensure the following tools are installed before getting started:

  • Yarn

    Make sure Yarn is installed and running on your system. Yarn is recommended over NPM.

  • Node.js

    Ensure Node.js version 16 or higher is installed.

  • Git

    Git is required for version control.

Installation

To set up the Venturin - Portfolio & Agency Tailwind CSS 4 Website Template, follow the steps below:

Command Description
yarn install Installs all required dependencies into the node_modules directory.
yarn dev Starts the local development server at http://localhost:5173 .
yarn build Generates the production-ready /dist folder.
yarn preview Previews the production build locally.

How to change the Font Family in Venturin?

You can update the font family by editing the theme configuration file. 

/* Theme Configuration */ in theme.css
@theme {
  --font-body: "DM Sans", sans-serif;
  ...
}
