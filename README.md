# Venturin - Portfolio & Agency Tailwind CSS 4 Website Template

Thank you for choosing **Venturin**! 😊  
This guide will help you get started with the template and its documentation.

## 📘 Docs and Examples

The `Documentation` folder contains all the static files required for usage examples and setup instructions.

To view the documentation, simply open the following file in your browser:

```bash
open Documentation/index.html
````

Once opened, click on the **Installation** link in the left sidebar.
This will load the `index.html` page containing step-by-step setup instructions.

## ⚡ Vite Setup

Venturin uses **Vite** for fast development and optimized production builds.
A preconfigured `vite.config.js` file is included for development and build automation.

### 🛠 Requirements

* **Node.js**
  Download: [https://nodejs.org/](https://nodejs.org/)

* **Vite**
  Documentation: [https://vitejs.dev/](https://vitejs.dev/)

### 📦 Installation & Build Commands

Run the following commands from the project root directory:

```bash
# Install dependencies
yarn install

# Start development server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview
```
